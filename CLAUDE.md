# 기술 스택

## Frontend
- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## Backend
- **API**: Next.js API Routes (동일 레포 모노레포 구조)

## Database
- **Provider**: Supabase (PostgreSQL + Auth)

## AI
- **SDK**: Anthropic SDK (`@anthropic-ai/sdk`)
- **분석 & 리액션**: `claude-haiku-4-5`
- **주간 요약**: `claude-sonnet-4-6`

## 배포
- **FE + BE**: Vercel
- **DB**: Supabase
